## Music Cover and Sharing Platform

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/37febaca51e94194a17d36e717adeab0/ghb20250917) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can be used to run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): regular users, administrators.

**Regular User Functions:**
1. Login
2. Registration
3. Homepage
4. Music News
5. Music Covers
6. Online Music Listening
7. Message Feedback
8. Personal Center
9. My Favorites
10. Customer Service Chat
11. Change Password
12. My Music Covers


**Administrator Functions:**
1. Login
2. Personal Center
3. User Management
4. Music News Management
5. Music Cover Management
6. Online Music Management
7. Message Board Management
8. Customer Service Management
9. Carousel Management
